# Store-Manager App
This App  allows the store manager to monitor the products that are sold by the store attendants and also add, modify or delete a product as he or she wishes.

##  Required Features
       
* Login page
* Signup page
* User profile pages 

##  Prerequisites
* html
* css
* javascript
